# Source blueprints

This directory contains the two blueprint documents that shaped PGSCP. They are kept verbatim (transcribed from the original PDFs into markdown) as reference material — the authoritative executable plan lives in [../plan.md](../plan.md), not here.

| File | Origin | Role |
|---|---|---|
| [cloud-engineer-project-blueprint.md](cloud-engineer-project-blueprint.md) | Initial project brief | High-level scope, milestones, interview narrative |
| [pgscp.md](pgscp.md) | Detailed follow-up blueprint | Authoritative technical design, IAM/Terraform sketches, incident playbook |

Where the two disagree, `pgscp.md` wins. The resolved decisions are captured in [../plan.md](../plan.md) and the ADRs in [../decisions/](../decisions/).
